################ HOW TO COMPILE ##############
- Open CMD
- Go to scanner ("dir", or "cd")
- Run command: java -cp javacc.jar jjtree ArduinoAST.jjt
- Run command: java -cp javacc.jar javacc ArduinoAST.jj
- Run command: javac -cp .;javacc.jar *.java (Windows) javac -cp .;javacc.jar *.java (FaggotOS)

######## HOW TO GET AST WHEN COMPILED ########
- Run command "java ArduinoAST"
- Enter code to get AST
